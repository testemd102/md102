40- ARRASTE E SOLTE -
Você tem um computador que executa o Windows 10 e contém dois usuários locais chamados Usuário1 e Usuário2.
Você precisa garantir que os usuários possam executar as seguintes ações:
O usuário1 deve ser capaz de ajustar a data e a hora.
O usuário2 deve ser capaz de limpar os logs do Windows.
A solução deve usar o princípio do menor privilégio.
A qual grupo você deve adicionar cada usuário?
Para responder, arraste os grupos apropriados para os usuários corretos.
Cada grupo pode ser usado uma vez, mais de uma vez ou nunca.
Talvez seja necessário arrastar a barra de divisão entre os painéis ou rolar para visualizar o conteúdo.
NOTA: Cada seleção correta vale um ponto.
